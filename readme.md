
# OJIX Browser

OJIX Browser is a fast, lightweight, and privacy-focused web browser built with Python.  
Designed for smooth performance, clean UI, and a secure browsing experience.

## Features
- Fast startup and smooth performance  
- Clean and modern UI  
- Lightweight Python-based engine  
- Safe and privacy-friendly  
- Frequent improvements and updates
